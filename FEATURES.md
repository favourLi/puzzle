# 功能详细说明文档

## 1. 图片选择功能

### 功能描述
用户可以选择本地图片作为拼图素材，图片仅在浏览器本地处理，不上传到服务器。

### 详细规格

| 项目 | 规格 |
|------|------|
| 支持格式 | JPG, PNG, WebP, GIF（仅第一帧） |
| 文件大小 | 最大 5MB |
| 图片尺寸 | 建议 300px ~ 2000px |
| 选择方式 | 点击选择本地文件 |

### 交互流程

```
1. 用户点击选择图片区域
      ↓
2. 打开文件选择对话框
      ↓
3. 选择图片后验证格式和大小
      ↓
4. 格式正确 → 加载图片，自动开始游戏
   格式错误 → 显示错误提示
```

### 本地处理说明

- 图片通过 `FileReader` API 读取到内存
- 使用 Canvas 在本地进行切割处理
- 切割后的图片块使用 Blob URL 缓存
- 整个过程无需服务器参与

### 异常处理

- 文件过大：提示"图片大小不能超过 5MB"
- 格式不支持：提示"请上传 JPG、PNG 或 WebP 格式的图片"
- 图片损坏：提示"图片文件损坏，请重新选择"

## 2. 难度选择功能

### 功能描述
提供三种难度模式，每种模式对应不同的拼图块数量。

### 难度配置

| 模式 | 网格 | 拼图块数 | 完成奖励积分 |
|------|------|----------|--------------|
| 简单 | 3×3 | 9 块 | 100 分 |
| 中等 | 4×4 | 16 块 | 200 分 |
| 复杂 | 6×6 | 36 块 | 500 分 |

### 交互规则

- 游戏未开始时可自由切换难度
- 游戏进行中禁止切换难度
- 默认选择"简单"模式
- 切换难度后需要重新开始游戏

---

## 3. 拼图游戏逻辑

### 功能描述
核心拼图玩法，用户通过拖拽拼图块来还原图片。

### 游戏规则

1. **初始状态**：拼图块随机打乱排列
2. **操作方式**：拖拽一个拼图块到另一个位置，两者交换
3. **完成条件**：所有拼图块回到正确位置

### 拼图块状态

```javascript
{
  id: 0,              // 唯一标识（正确位置索引）
  currentIndex: 3,    // 当前位置索引
  correctIndex: 0,    // 正确位置索引
  imageUrl: 'blob:...' // 拼图块图片 URL
}
```

### 视觉反馈

- **拖拽中**：拼图块半透明，显示拖拽阴影
- **目标位置**：高亮显示目标格子
- **正确位置**：可选显示绿色边框提示
- **完成时**：播放庆祝动画

---

## 4. 计时功能

### 功能描述
记录玩家完成拼图所花费的时间。

### 计时规则

- 点击"开始游戏"后立即开始计时
- 完成拼图时停止计时
- 格式：`MM:SS`（分:秒）
- 最长计时：99:59

### 显示位置

- 游戏界面顶部
- 完成后的成绩面板

---

## 5. 移动次数统计

### 功能描述
记录玩家的操作次数，每次交换计为一次移动。

### 统计规则

- 点击"开始游戏"后清零
- 每次有效交换 +1
- 无效操作（拖到原位置）不计
- 完成时显示总移动次数

---

## 6. 预览功能

### 功能描述
随时查看原始完整图片作为参考。

### 交互设计

- 点击"预览"按钮显示原图
- 再次点击或点击空白处关闭
- 弹窗居中显示，带半透明遮罩
- 游戏计时不受影响

---

## 7. 重新开始功能

### 功能描述
使用当前图片和难度重新开始游戏。

### 重置内容

- 拼图块重新打乱
- 计时器归零
- 移动次数归零
- 保持当前图片和难度设置

---

## 8. 完成庆祝动画

### 功能描述
拼图完成时显示庆祝效果和成绩。

### 显示内容

| 项目 | 说明 |
|------|------|
| 完成时间 | 花费的总时间 |
| 移动次数 | 总操作次数 |
| 难度等级 | 当前游戏难度 |
| 评价 | 根据表现给出星级评价 |

### 评价标准

| 难度 | ⭐⭐⭐ | ⭐⭐ | ⭐ |
|------|-------|------|-----|
| 简单 | < 30s | < 60s | ≥ 60s |
| 中等 | < 90s | < 180s | ≥ 180s |
| 复杂 | < 180s | < 300s | ≥ 300s |

### 动画效果

- 拼图块闪烁发光
- 彩带/烟花效果
- 成绩面板滑入

---

## 9. 选择新图片

### 功能描述
放弃当前图片，选择新图片开始游戏。

### 交互流程

```
1. 点击"选择图片"按钮
      ↓
2. 打开文件选择对话框
      ↓
3. 选择新图片
      ↓
4. 重置游戏状态，开始新游戏
```
