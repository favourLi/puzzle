# 拼图游戏 Web 应用

## 项目简介

这是一个基于 Vue 3 的拼图游戏应用，用户可以选择本地图片，将其打散成拼图块进行游戏。图片仅在浏览器本地处理，无需上传服务器。应用提供三种难度模式，适合不同水平的玩家。

## 功能特性

- **本地图片选择** - 选择本地图片作为拼图素材，无需上传服务器
- **三种难度模式** - 简单(3x3)、中等(4x4)、复杂(6x6)
- **拖拽交互** - 直观的拖拽操作，流畅的游戏体验
- **计时功能** - 记录完成拼图所用时间
- **移动次数统计** - 跟踪玩家的操作次数
- **预览功能** - 随时查看原始图片作为参考
- **重新开始** - 一键重新打乱拼图
- **完成动画** - 拼图完成时的庆祝效果

## 技术栈

| 技术 | 版本 | 用途 |
|------|------|------|
| Vue | 3.x | 前端框架 |
| Vite | 5.x | 构建工具 |
| Pinia | 2.x | 状态管理 |
| CSS3 | - | 样式设计 |

## 难度模式

| 模式 | 网格 | 拼图块数 | 适合人群 |
|------|------|----------|----------|
| 简单 | 3×3 | 9 块 | 新手玩家、儿童 |
| 中等 | 4×4 | 16 块 | 有一定基础的玩家 |
| 复杂 | 6×6 | 36 块 | 挑战型玩家 |

## 用户操作流程

```
1. 打开应用
      ↓
2. 选择难度模式（默认简单）
      ↓
3. 选择本地图片或使用默认图片
      ↓
4. 图片自动加载，开始游戏
      ↓
5. 拖拽拼图块进行拼图
      ↓
6. 完成拼图，显示成绩
      ↓
7. 可选择重新开始或选择新图片
```

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 项目目标

1. 提供流畅的拼图游戏体验
2. 支持自定义图片增加趣味性
3. 多难度模式满足不同玩家需求
4. 记录成绩增加挑战动力
